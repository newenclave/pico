## Very simple and small interpreted language implemented with python

### Supports

numbers, strings, arrays, tables, functions as first class citizens

```
    let a = 100
    let b = 200
    let sum = fn(x, y) { x + y }
    let c = sum(a, b)
    let s = "This is a string"
    let t = {true: "True", false: "False"}
```
